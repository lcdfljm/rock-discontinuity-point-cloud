# Rock Discontinuity Point Cloud Dataset

This repository provides a point cloud dataset acquired from an artificially excavated highway slope in Ouray, Colorado, USA. The dataset can be used for rock mass discontinuity identification, discontinuity-set classification, and point cloud segmentation.

## Dataset

The complete point cloud dataset is available in Release v1.0.

Release asset:

- `Ouray_point_cloud_dataset.rar`

## Data description

The dataset contains point cloud data of an artificially excavated highway slope. The point cloud can be used to test clustering-based methods for identifying rock mass discontinuities.

## Original data source

The Ouray point cloud dataset used in this repository was originally obtained from the public RockBench repository. The original data source was provided at:

`www.3D-landslide.com/projects/discontinuity/`

However, the original link is currently unavailable. To ensure data accessibility and reproducibility, the dataset used in this study has been re-uploaded to this GitHub repository and is available in Release v1.0 as `Ouray_point_cloud_dataset.rar`.

This repository is only used for academic research and data reproducibility.

## File format

The released dataset is compressed as a `.rar` file. After decompression, the point cloud data can be opened and processed using common point cloud software such as CloudCompare.
